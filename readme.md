# Hacker Pouch
A Hacker news clone using Angular 1.5 and PouchDB and [HackerNews API](https://github.com/HackerNews/API). 

### To run
To get started, run `npm install & npm run dev-serve`.  This will install all node_modules and start
* node server
* node-sass
* watchify
